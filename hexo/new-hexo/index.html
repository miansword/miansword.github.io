2015-05-19hexo


最近使用了Hexo建了自己的一个博客,使用过程中，Hexo给人感觉是一种舒畅感，一气呵成的，复制粘贴几条便轻松轻松地建立起自己的博客。而且，Hexo是一个静态博客系统，可以省去了配置数据库的问题，再利用Github Pages服务,有省去了挑选空间的时间更空间租用费用，基本可以零成本建立起一个属于自己的博客了。

HEXO到时是啥

Hexo is a fast, simple & powerful blog framework powered by Node.js.

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

使用后感

优点

  • 网站配置过程中还是相当便利的，几条简单命令，一些简单的配置（复制粘贴），基本就可以生成一个比较不错的站点了（当然这得益于Hexo主题啦！）
  • 自带Markdown写作格式，让博主更方便地创造内容，已不用再关注排版，更抛离了传统的后台内容写作方式，随便一个编辑器就可以创作你的神文啦。不过还是需要懂得基本的Markdown格式，不然没有效果哦。

缺点

  • 没有了传统网站后台编辑器的便利，添加文章后，需要寻找相应的md文件去修改内容，才能显示到内容。
  • 所用添加文章，生成内容等等，没有了人性化的UI界面，全部需要运行命令来解决，建站入门门槛相对的高。
